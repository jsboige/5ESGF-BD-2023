# Sudoku Solver : GeneticSharp

Réaliser par : PONGUI Beaudross et MESSAOUDI Rihab 

Nous avons choisi le solver GeneticSharp car c'est une librairie qui utilise des algorithmes génétique s'inspirant de la théorie de l'évolution de Darwin.C'est une méthode d'optimisation
pour l'auto-organisation et l'auto-adaptation concernant la simulation du processus d'évolution des espèces dans la nature et dans le monde.


## Ce solver contient plusieurs classe : Chromosome,  ChromosomeBase, Fitness, Cell Chromosome:



Nous retrouvons des fonctions comme **SudokuCellsChromosome()**: Initialise un nouvel objet chromosomeCell avec une grille Sudoku spécifiée. 
**GenerateGene()** est un méthode pour générer un génome initiale ou aléatoire à ses début pour la création de population. **Creategene()** est utiliser pour créer
un nouvel objet de gène avec une valeur approprié.
